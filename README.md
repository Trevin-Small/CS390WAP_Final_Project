### Fall 2023 | CS390 - Web App Programming
Recipe Website - Final project

## Project Setup
`cd` into both `/client` and `/server` directories, run:
`npm install`

## Run The Website (Note you must do both of these, in this order)
**1. Server**
`cd server`
'npm start'

**2. Client**
`cd client`
`npm start`

**Visit Page**
`http://localhost:3000`
