# Yummy Yields - Recipe Website
### CS390 - Web App Programming Final Project | Fall 2023

# Run The Project
**0. Install dependencies in client and server directories**  
`npm install`  

**1. Start the server**  
`cd server`  
`npm start`  

**2. Start the client**  
`cd client`  
`npm run build`
`npm start`  

**3. Run Tailwind (for development only)**  
`cd client`  
`npm run tailwind`  

**Visit the page**  
`http://localhost:3000`
